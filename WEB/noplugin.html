Welcome to Webvue !

The Java Platform Plug-in is not available. It has either not been registered with your web browser or it is not installed on your system.
This component is required to run WebVue.
Before the first use of WebVue the Java Runtime Environment (JRE) needs to be installed.

The Java Runtime Environment is available:

  • On the Supervisor’s distribution media. (32-bit version only)
    • %COMMENT_START%
  • As a download from the WebVue Server (32-bit version only)
    • %COMMENT_END%
  • As a download from Oracle's Web Site

As soon as the Java Plug-in is installed, please click on the Refresh button.

If you are still experiencing problems please refer to the Supervisor’s help or contact your support agency.
